# Study Dev Break Timer

A minimalist focus timer built for developers, students, and deep work sessions.

## Features

- Study / Work / Break presets
- Custom timer durations
- Persistent local storage settings
- Alarm sound support
- Vibration support
- Clean dark UI
- Responsive layout
- Built with React + Vite + Tailwind CSS

## Tech Stack

- React
- Vite
- Tailwind CSS

## Run Locally

```bash
npm install
npm run dev
